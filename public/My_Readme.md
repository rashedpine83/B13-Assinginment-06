 Name:  DigiTools

Project Description:

DigiTools is a modern web application designed to supercharge your digital workflow. It provides access to premium AI tools, design assets, templates, and productivity software—all in one place to help users create faster and more efficiently.

 Technologies Used:
    React.js
    HTML
    JavaScript
    Tailwind CSS
    DaisyUI
    React Icons
    React-Toastify (NPM Package)
    JSON (for product data)



Features:
    1. Responsive Design — Works perfectly on mobile, tablet, and desktop devices.
    2. Interactive UI — Clean and modern interface with smooth user experience.
    3. Dynamic Cart System — Real-time cart updates with item count badge.
